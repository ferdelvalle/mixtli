About Mixtli

Mixtli is a platform built to showcase data in the shape of charts created for this project, it was built using rather simple technologies such as Zoho Forms and Google Sheets in order to display the data gathered for the project in a simple and friendly way as part of "The lack of adoption of Cloud Computing in Latin America" project.

This project aims to find out what causes the lackluster adoption rates for Cloud Computing Technologies in Latin America through a survey answered by several organizations. The data gathered is shown in the Data section.

Mixtli is an open source platform and you can find the link to the source code at the bottom or in the About section.
